# CP Problem Template

Template này dùng để phát triển bài toán cho lập trình thi đấu (competitive programming) cho ICPC, OI, contest nội bộ, soạn bài, v.v..

Nó cung cấp đầy đủ công cụ để:
- viết lời giải mẫu
- tạo test tự động
- kiểm tra input
- viết output checker
- stress test lời giải

Cấu trúc chính nằm trong thư mục [`template/`](template/).
Hướng dẫn sử dụng cơ bản nằm trong thư mục [`docs/`](docs/).
